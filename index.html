A propos de moi

Après avoir effectué l'ensemble de mes études dans le domaine bancaire et exercé mon activité de conseiller notamment sur le marché des professionnels, j'ai finalement réalisé que ce cadre ne me correspondait pas de par le manque de flexibilité et l'approche client de celui-ci. Ayant toujours eu une curiosité jamais exploitée pour ce secteur d'avenir qu'est le domaine du numérique je me lance aujourd'hui dans une reconversion totale : Le développement web. Mon objectif est d'obtenir les compétences nécessaires afin de rejoindre une entreprise qui a coeur d'accompagner nos entreprises françaises dans leur transition et stratégie numérique. Pour se faire, je rejoins dès le 28 février la WILD CODE SCHOOL afin d'avoir toutes les cartes en main pour débuter cette aventure.
